桌面端

添加快捷方式到桌面：

通用：桌面点右键 → 新建 → 快捷方式 → 输入：http://www.sztnet.com/ → 下一步 → 输入：上证通网。

谷歌：浏览器右上角省略号 → 更多工具 → 添加到桌面... → 输入“上证通网”（不要勾选“在窗口打开”）。

设为浏览器首页：

谷歌：浏览器右上角省略号 → 设置 → “外观”-“显示主页按钮” → 第二选项中输入“www.sztnet.com/”即完成。使用时点击左上角"房子"图标即可访问。

IE：浏览器右上角齿轮图标 → Internet选项 → “常规-主页-使用当前主页” → “确定”即完成。防止杀毒软件阻止。

360：浏览器右上角三道杠 → 设置 → “外观”-“显示主页按钮” → 第二选项中输入“www.sztnet.com/”即完成。使用时点击浏览器左上角的主页图标（地址栏前）即可访问。
